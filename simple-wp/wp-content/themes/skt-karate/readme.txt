== Description ==

SKT Karate is a martial arts, taekwondo, kung fu, MMA type of theme. Responsive, WooCommerce friendly. Comes with 150+ other design alternatives. Inner pages compatible with Gutenberg editor. Other industries which can use this template are fitness, sports, gym, body building, instructor, activity, health experts, bootcamp, diet planner, yoga, workout, power crossfit, physiotherapy, wellness, aerobics, kick boxing, cardio etc. It is a core web vitals optimized theme. Works well with cache and SEO plugins.
